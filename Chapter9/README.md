5.  PM 和一帮开发人员合作，PM 和开发领导 （dev lead）是什么关系，听谁的？
```
- 任务的重要、紧急程度需要听PM的
- 可行性、开发时间评估需要听Dev Lead的
- 风险控制要听PM的，但开发过程的风险需要听DevLead的意见，PM的其他风险也需要告知DevLead
- 当和其他Team冲突时，需要听PM的，但Dev Lead需要给出Dev角度的意见
所以他们的关系式合作关系
```
